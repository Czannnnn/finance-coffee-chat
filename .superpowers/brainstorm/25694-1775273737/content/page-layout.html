분석 페이지 레이아웃

페이지의 전체 구성을 어떤 방식으로 할까요?

A

테이블 중심 + 행 클릭 확장

전체 종목을 한눈에 비교하는 테이블 → 행 클릭 시 상세 분석 패널 확장

┌─────────────────────────────────────────────────────┐
│ 🔍 등급 필터: [전체] [S] [A] [B] [C/D]   상태 필터 │
├──────┬────────┬────────┬──────┬──────┬──────┬──────┤
│ 등급 │ 종목명 │ 경쟁률 │공모가│ 확약 │유통량│ 일정 │
├──────┼────────┼────────┼──────┼──────┼──────┼──────┤
│  S  │ AAA   │ ✅1200│ ✅상단│ ✅52%│ ✅18%│ 4/7~ │
│ ▼ 상세 펼침: 투자 포인트, 리스크, 섹터 분석      │
├──────┼────────┼────────┼──────┼──────┼──────┼──────┤
│  B  │ BBB   │ ❌ 400│ ✅상단│ ❌ 8%│ ❌32%│ 4/10~│
└──────┴────────┴────────┴──────┴──────┴──────┴──────┘

장점

  • 종목 간 빠른 비교
  • 데이터 밀도 높음
  • PC에서 최적

단점

  • 모바일에서 테이블 스크롤 필요
B

카드 리스트 (등급별 그룹핑)

등급(S→D)별로 그룹핑된 카드 리스트. 각 카드에 4대 조건 게이지 표시.

── S등급 (적극 추천) ──────────────────────
┌─────────────────────────────────────────┐
│ 🏆 AAA주식회사          코스닥 │ 청약중 │
│ ① 경쟁률 1,200:1 ✅ │ ② 공모가 상단 ✅ │
│ ③ 확약 52%     ✅ │ ④ 유통 18%   ✅ │
│ 📅 4/7~4/8 │ 🏦 한국투자증권       │
└─────────────────────────────────────────┘

── A등급 (추천) ──────────────────────────
┌─────────────────────────────────────────┐
│ ...카드...                              │
└─────────────────────────────────────────┘

장점

  • 모바일 친화적
  • 등급별 시각적 구분 명확
  • 기존 커피 테마 카드와 통일감

단점

  • 종목 간 수치 비교가 테이블보다 어려움
C

하이브리드: 요약 테이블 + 카드 상세

상단에 간결한 요약 테이블(등급/종목/4조건 O/X)로 전체 비교 → 하단에 카드형 상세 분석

┌ 요약 테이블 (모바일: 가로스크롤) ─────────┐
│ S│AAA │ ✅✅✅✅ │ 4/7~ │ 클릭→상세 │
│ B│BBB │ ❌✅❌❌ │ 4/10~│ 클릭→상세 │
└──────────────────────────────────────────┘
                ↓ 스크롤 ↓
┌ 카드 상세 ───────────────────────────────┐
│ 🏆 AAA주식회사               S등급 │
│ 투자 포인트 / 리스크 / 섹터 분석       │
└──────────────────────────────────────────┘

장점

  • 빠른 비교 + 깊은 분석 모두 가능
  • PC/모바일 모두 대응

단점

  • 페이지 길이가 길어짐
  • 구현 복잡도 높음

추천: Option B — 기존 커피 테마 카드 디자인과 통일감 있고, 모바일 친화적. 등급별 그룹핑으로 직관적 탐색.
