공개 페이지 — 콘텐츠 범위

GEM 스펙에서 공개 페이지에 적합한 요소를 선별합니다. 아래 구성이 맞는지 확인해 주세요.

✅ 포함할 요소

  • 필승 전략 4대 조건 분석 프레임워크 — 기관 경쟁률 800:1↑ / 공모가 상단 초과 / 의무 확약 15%↑ / 유통 물량 25%↓
  • 등급 시스템 (S/A/B/C/D) — 교육적 참고 지표로 제공
  • 종목 기본 정보 — 종목명, 시장(코스피/코스닥), 청약일정, 주관사
  • 4대 조건 수치 — 기관 경쟁률, 확정 공모가(밴드 대비), 의무 확약 비율, 유통 물량 비율
  • 등급 가이드 안내 섹션 — S~D 등급 기준 설명
  • 면책 조항 — "투자 참고용 자료이며, 투자 판단의 책임은 본인에게 있습니다"

❌ 제외할 요소 (개인 투자 정보)

  • 투자 추천 여부 컬럼 — "추천/비추천" 직접 표현 제외
  • 비례 배정 & 자금 운용 가이드 — 개인 자금 관리 영역
  • 5사 6입 최적화, 증거금 구간 — 개인 투자 전략
  • 매도 전략 / 목표가 — 개인 매매 판단
  • 구글 시트 자동화 명령어 (패턴 A/B/C) — 개인 도구
  • 정기 보고 / 수시 보고 체계 — GEM 에이전트 전용
  • "치킨 값 확보" 등 개인 톤 — 공개 페이지에 부적합

💡 공개 버전으로 변환

GEM 원본 → 공개 페이지 변환

공모주린이 등급 → "분석 등급"으로 명칭 변경 (채널 참조 제거)
비례 배정 가이드 → 제외 (4대 조건 데이터만 제공)
커뮤니케이션 스타일 → 중립적 데이터 중심 톤
