자주 묻는 질문

5분 자동 갱신은 어떻게 작동하나요?

본 페이지는 setInterval로 5분마다 Google Sheets normalized 시트를 다시 fetch하여 변경된 데이터를 반영합니다. 페이지를 켜둔 채로 두면 별도 새로고침 없이 최신 데이터가 표시되며, 라이브 변경 카운트(이번 세션 동안 변경된 종목 수)가 우상단에 표시됩니다.

freshness 배지의 의미는?

마지막 데이터 갱신 시각과 현재 시각의 차이를 색상 배지로 표시합니다. 🟢 1시간 미만: 매우 신선 / 🟡 1~6시간: 보통 / 🔴 6시간 이상: 오래된 데이터(데이터 파이프라인 트리거 06:00·08:00·16:00·18:00 KST 사이 자연 발생).

upcoming과 pending_listing의 차이는?

upcoming은 청약 시작일이 오늘 이후인 종목, pending_listing은 청약은 끝났으나 상장일 도래 전인 종목입니다. 본 페이지는 두 그룹을 분리 표시해 일정 추적을 용이하게 합니다.

데이터 신뢰도 confidence는?

high = DART OpenDart API document.xml 본문 매칭 OK, medium = DART corp_code 매칭 + 보조 공시 데이터로 채움, flag = 출처 간 ±10원 이상 불일치(파싱 오류 의심). flag 종목은 별도 표시되며 청약 결정 전 투자설명서 직접 확인 권고.

왜 모바일 카드가 작은가요?

실시간 페이지는 다수 종목을 한 화면에 표시하기 위해 콤팩트 행 레이아웃을 사용합니다. 종목별 상세 정보(4대 조건 등급 분석)는 /ipo-analysis 페이지에서 확인할 수 있고, 청약 절차는 /ipo-guide에서 상세 가이드를 제공합니다.
