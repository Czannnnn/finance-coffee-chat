자주 묻는 질문

오늘 청약 직전·직후 가장 많이 받는 질문입니다.

오늘 청약 마감 시각은 언제인가요?

증권사·종목별로 다르지만 일반적으로 청약 마지막 날 16:00이 마감입니다. 일부 증권사는 17:00 마감(NH투자·KB증권 등)·14:00 마감(소형 증권사)도 있어 청약 신청 화면의 '청약 마감 시각'을 직접 확인해야 합니다. 본 페이지의 카운트다운은 16:00 기준입니다.

오늘 청약 마감인 종목인데 증거금은 언제 입금해야 하나요?

청약 마감 시각(16:00 기준) 이전에 증거금이 계좌에 들어와 있어야 합니다. 청약 시점 잔고 부족이면 청약 자체가 거절됩니다. 마감 임박 시 다른 증권사로 이체에 5~10분 소요될 수 있으니 마감 30분 이상 전에 신청 완료를 권합니다.

청약 환불금은 언제 받나요?

환불은 청약 마감 다음 날(영업일 기준 +1일)이 표준입니다. 본 페이지의 각 종목 카드 '환불일'을 확인하세요. 균등 배정 만으로도 증거금이 회수되므로 단기 자금 활용 계획 시 환불일 정확히 체크 권장.

균등 배정과 비례 배정 차이는?

균등은 최소 청약수량 이상 신청한 모든 청약자에게 동일하게 배정. 비례는 신청 주식수 비례 배정. 일반적으로 50:50 분할이며 균등 배정은 1주~10주 수준이 흔합니다. 자세한 절차는 공모주 청약 가이드 페이지에 단계별 정리되어 있습니다.

데이터 갱신 주기는?

Apps Script가 DART OpenDart API 공시 자료를 일 4회(06:00·08:00·16:00·18:00 KST) 동기화하여 normalized 시트에 적재합니다. 본 페이지는 시트의 gviz API로 페이지 로드 시 즉시 fetch하며, 페이지 우상단 freshness 배지(< 1h 🟢 / < 6h 🟡 / < 24h 🔴)로 마지막 갱신 시각을 표시합니다.
