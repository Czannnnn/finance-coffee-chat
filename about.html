무엇을 하는 곳인가

투자 커피챗(Finance Coffee Chat)은 한국 개인투자자를 위한 금융 교육·정보 콘텐츠 플랫폼입니다. 가입 없이 바로 쓰는 도구와 읽기 쉬운 가이드로, 공모주·지수투자·자산관리·절세·생활비 최적화를 다룹니다.

  • 공모주 캘린더·분석 — DART 공시 기반 청약 일정과 4대 조건 등급
  • 자산관리 101 — 입문자를 위한 단계별 자산배분 교육
  • 절세·연금 가이드 — ISA·연금저축·IRP 등 제도 해설
  • 계산기·시뮬레이터 — 세금·비용·포트폴리오를 직접 계산

이 사이트는 직접 콘텐츠를 발행하는 1인 운영 퍼블리셔(first-party publisher)입니다. 제3자의 광고를 대행하거나 운영하지 않으며, 특정 종목 매수 권유가 아닌 교육·정보 제공을 목적으로 합니다.

데이터를 어떻게 활용하나 (콘텐츠 기획)

독자에게 더 필요한 주제를 먼저 쓰기 위해, 키워드 검색량 데이터를 활용합니다.

어떤 글을 다음에 쓸지 정하기 위해, 한국 시장의 키워드 검색량·연관 키워드 데이터를 조회해 콘텐츠 우선순위를 정합니다. 두 가지 공식 데이터 소스를 사용합니다.

  • 네이버 검색광고 키워드도구 — 한국어 키워드의 월간 검색량 조회
  • Google Ads 키워드플래너 (Google Ads API) — 동일 키워드의 구글 검색량·연관 키워드 조회

이 데이터는 오직 본 사이트 자체의 SEO 콘텐츠 기획에만 쓰입니다. 읽기 전용(read-only) 조회이며, 광고 캠페인을 생성·수정·운영하지 않고, 데이터를 외부에 재판매하거나 제3자에게 제공하지 않습니다. 도구는 운영자 1인이 내부적으로만 사용합니다.

For reviewers · How we use the Google Ads API

Finance Coffee Chat (financecoffeechat.com) is a first-party Korean personal-finance content website. We publish educational articles and free interactive calculators (IPO calendar, tax, pension/IRP, ISA, ETF investing). We do not manage ads for any third party.

We use an internal, single-user tool that calls the Google Ads API KeywordPlanIdeaService (Generate Keyword Ideas / Historical Metrics) to retrieve search-volume and related-keyword data for the Korean market. This data is used solely to plan and prioritize SEO content for this website.

The usage is read-only (we never create, modify, or manage ad campaigns), internal (only the operator uses it; no external interface, no resale, no third-party access), and low volume (a few dozen keyword queries per week).
