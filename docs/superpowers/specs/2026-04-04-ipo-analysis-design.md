# 공모주 분석 페이지 설계

**작성일:** 2026-04-04
**페이지:** `ipo-analysis.html`
**상태:** 설계 승인 완료

---

## 1. 목적

공모주 종목별 "필승 전략 4대 조건" 기반 분석을 제공하는 공개 페이지.
개인 투자 정보(추천 여부, 자금 운용, 매도 전략 등)를 제외하고, 데이터 중심의 중립적 분석 프레임워크를 제공한다.

## 2. 아키텍처 결정

| 항목 | 결정 | 근거 |
|------|------|------|
| 페이지 구조 | 별도 페이지 (`ipo-analysis.html`) | 캘린더 복잡도 유지, URL 공유 가능 |
| 레이아웃 | 카드 리스트 + 등급별 그룹핑 | 모바일 친화, 기존 커피 테마 통일 |
| 데이터 소스 | 샘플 데이터 우선 → Google Sheets 연동 (후속) | 기존 캘린더와 동일 패턴 |
| 프레임워크 | 바닐라 HTML/CSS/JS | 기존 사이트와 동일 |

## 3. 4대 조건 프레임워크

| # | 조건 | 기준 | 충족 판정 |
|---|------|------|-----------|
| ① | 기관 경쟁률 | 800:1 이상 | 숫자 비교 |
| ② | 확정 공모가 | 희망 공모가 밴드 상단 이상 | 확정가 >= 밴드 상단 |
| ③ | 의무 확약 | 15% 이상 | 숫자 비교 |
| ④ | 유통 물량 | 25% 이하 | 숫자 비교 |

## 4. 등급 시스템

| 등급 | 기준 | 색상 |
|------|------|------|
| S | 4대 조건 완벽 충족 + 확약 50% 이상 | Green (#4A7C59) |
| A | 3개 이상 충족 + 섹터 분위기 양호 | Blue (#3D5A80) |
| B | 2개 충족 | Caramel (#C4853C) |
| C | 1개 충족 | Red-soft (#C0564B) |
| D | 0개 충족 | Red-soft (#C0564B) |

등급은 샘플 데이터에 수동 지정. 향후 시트 연동 시 시트에서 관리.

## 5. 페이지 구성

### 5-1. 헤더
- 커피 테마 배지 "IPO Analysis"
- 타이틀: "공모주 분석"
- 서브타이틀: "필승 전략 4대 조건 기반 종목 분석"
- 데이터 소스 배지 (SAMPLE DATA / LIVE DATA)

### 5-2. 4대 조건 가이드 카드
- 4열 그리드 (모바일 2열)
- 각 조건: 아이콘 + 조건명 + 기준값

### 5-3. 등급 가이드 카드
- S/A/B/C/D 등급별 기준 설명
- 등급 배지 색상 미리보기

### 5-4. 필터
- 등급 필터: 전체 / S / A / B / C/D
- 상태 필터: 전체 / 청약예정 / 청약중 / 상장완료

### 5-5. 등급별 카드 리스트
- 등급 섹션 헤더 (예: "S등급 — 4대 조건 충족")
- 각 카드 구성:
  - **상단**: 등급 배지 + 종목명 + 시장 배지 + 상태 배지
  - **중단**: 4대 조건 그리드 (4열, 모바일 2열)
    - 각 조건: 조건명 + 수치 + ✅/❌ + 기준값 참고
  - **하단**: 청약일정, 주관사, 공모가 밴드, 공모주식수

### 5-6. 면책 조항
- "본 페이지의 분석 등급 및 데이터는 투자 참고용 자료이며, 투자 판단의 책임은 본인에게 있습니다."

## 9. 제외 항목 (공개 페이지)

- 투자 추천/비추천 직접 표현
- 비례 배정 & 자금 운용 가이드
- 5사 6입 최적화, 증거금 구간
- 매도 전략 / 목표가
- 구글 시트 자동화 명령어
- 정기/수시 보고 체계
- 개인 톤앤매너 ("치킨 값 확보" 등)
- "공모주린이" 명칭 → "분석 등급"으로 중립화
