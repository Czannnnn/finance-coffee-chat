# Cycle 3 리서치 설계 — finance-coffee-chat 자체 수익 잠재력

**작성일**: 2026-04-14
**작성자**: 브레인스토밍 세션 (여태경 × 조정자)
**상태**: 승인 대기
**이전 사이클**: Cycle 1 (포지셔닝, 2026-04-13), Cycle 2 (콘텐츠 제작 방향성, 2026-04-14)

---

## 1. 배경

Cycle 1과 Cycle 2는 모두 **여태경 개인 브랜드(한국판 FIRE 블로거)**를 전제로 한 리서치였다. Cycle 1은 포지셔닝·빈자리·무기를, Cycle 2는 콘텐츠 제작 방향성(필러·포맷·편집 룰)을 각각 확정했다.

Cycle 3은 이 전제를 의도적으로 제거한다. 즉 **fcc 사이트가 독립 제품으로서** 얼마나·어떻게 돈을 벌 수 있는지를 평가한다. 개인 브랜드 후광을 0으로 두고 "사이트 자체의 수익 잠재력"을 측정하는 것이 목표다.

## 2. 핵심 질문

> **fcc가 여태경 개인 브랜드 후광 없이 독립 사업으로 성립하는가? 성립한다면 1년/3년 기준 얼마나·어떻게 벌 수 있나?**

이 질문은 세 층위를 통합한다:

- **A. 할지 말지** — 독립 사업으로 성립 가능한가? 성립하면 본격 투자, 아니면 Cycle 1/2의 개인 브랜딩 보조물로 남긴다
- **B. 얼마나 벌 수 있나** — 현실·낙관·비관 시나리오의 월매출 추정
- **C. 어떻게 벌까** — 광고·유료 구독·어필리엇·SaaS 툴·데이터 API·유료 PDF·강의·커뮤니티·스폰서십 중 무엇

## 3. 조사 전제

| 항목 | 설정 |
|---|---|
| **현 자산** | 공모주 DB (IPO 4기준 스코어링), 비용 최적화 (Phase 9 6×8 카탈로그 + 개인화 필터), 투자 캘린더, fcc 도메인, GA4 태깅, Vercel 배포 |
| **확장 허용** | 점진 확장 baseline. 조사 결과에 따라 피벗도 옵션 |
| **시간 지평** | 1년 보수 / 3년 현실 / 3년 낙관 세 시나리오 병행 |
| **트래픽 데이터** | 현재 의미 없음 → 한국 벤치마크로 앵커 |
| **운영 역량** | 10h/week 1인 운영 vs 2~3인 확장 두 시나리오 세트 분리 |
| **벤치마크 지리** | 한국 중심 |
| **법적 가드레일** | Cycle 2의 2024 개정 자본시장법 결론 그대로 적용 (유료 리딩방·개별 추천 금지선, 무료+교육+불특정다수는 안전) |

## 4. 성공 기준

리포트를 읽고 여태경이 다음 결정을 내릴 수 있어야 한다:
1. fcc를 독립 사업으로 본격 투자할 가치가 있는가?
2. 투자한다면 어느 수익 모델부터 시작해야 하는가?
3. 10h/week 제약을 풀고 풀타임·공동창업자·외주로 확장할 가치가 있는가?
4. 피벗이 필요하다면 어느 방향인가?

근거 기준: "추정이지만 출처 있는 숫자" 수준. 모든 추정치에 "공개치 / 업계 추정 / 가설" 라벨 강제.

## 5. 조사 접근법

**채택**: 4축 병렬 에이전트 (Cycle 1/2 검증된 워크플로 + Unit Economics 테이블과 TAM 간이 추정을 각 축 안으로 흡수)

**대안 고려**:
- TAM→SAM→SOM 퍼널 (기각: 한국 TAM 공개 데이터 희박, 병렬 불가)
- Unit Economics 우선 (기각: 한국 특화 벤치마크 희박. 단 테이블은 축 C에 흡수)

## 6. 4개 리서치 축

### 축 A — 한국 유사 사이트 수익 모델 인벤토리

**타겟**
- 토스피드 / 토스증권
- 뱅크샐러드 블로그
- 어피티 (UPPITY)
- 38커뮤니케이션 (공모주 직접 경쟁)
- 아이투자 itooza (공모주·가치투자 데이터)
- 네이버 프리미엄 콘텐츠 (스노볼 등)
- 한경 멤버스
- 슈카월드
- 김단테 이루다투자
- 신사임당 오너클랜 (참고)

**조사 질문**
- 각 사이트의 수익 모델 구성(광고/구독/어필리엇/강의/데이터 등)과 비중
- 공개된 월/연 매출·구독자 수
- 결정적 수익 엔진 1~2개
- 공모주 / 비용 최적화 / 투자 캘린더 세 니치와 직접 연결되는 사이트 **3곳 심층 분해**

**출력**
- 모델별 비교표 (타겟 × 수익원 × 규모 × ARPU 추정)
- fcc에 이식 가능성 점수 (0~5 Likert)
- **마지막 섹션**: 공모주 / 비용 최적화 / 투자 캘린더 각 니치의 한국 TAM 간이 추정 (월 검색량·관심 유저 수)

**분량**: ≤800 단어

### 축 B — 수익 모델별 한국 KPI 벤치마크

**조사 대상 모델과 KPI**
- **디스플레이 광고**: 구글 애드센스 한국 CPM·CPC, Taboola·Outbrain 단가
- **네이버 프리미엄 콘텐츠**: 입점 수수료·ARPU·월 구독료
- **뉴스레터 유료 전환**: 스티비·메일리 한국 평균 전환율, 월 구독료 (벤치: 롱블랙 4,900 / 조선멤버십 5,900 / 한경멤버스 1~3만)
- **강의**: 클래스101·인프런·패스트캠퍼스 수수료 구조·강사 평균 매출
- **어필리엇**: 쿠팡파트너스, 카드고릴라·핀크 등 금융 어필리엇 단가
- **데이터 API·B2B**: 한국 금융 데이터 서비스 단가 (크래프트테크·에프앤가이드 참고)
- **유료 PDF·템플릿**: 크몽·탈잉 가격대

**조사 질문**
- 각 모델의 한국 시장 unit economics (단가, 전환율, retention, CAC/LTV 대리지표)
- 공개치 없는 경우 업계 추정 범위와 출처

**출력**
- 모델 × KPI 매트릭스 (보수/현실/낙관 3구간)
- 데이터 공백 셀은 "가설" 라벨 + 근거 추정 방법 명시

**분량**: ≤800 단어

### 축 C — fcc 3자산에 모델 매핑 + 수익 시나리오 추정

**자산 정의**
| 자산 | 현재 상태 |
|---|---|
| **공모주 DB** | IPO 4기준 스코어링, 종목 상세, 캘린더 연동 (정적 JSON + Google Apps Script) |
| **비용 최적화** | Phase 9 6×8 카탈로그 + 2026-04-13 개인화 필터 + 48 세그먼트 × 영역 계산기 |
| **투자 캘린더** | IPO 일정, 배당락, 이벤트 캘린더 페이지 |

**모델 매핑 작업**
자산 × 수익 모델 = 적합도 평가 (0~5)
적용 후보 모델: 광고 / 유료 구독 / 어필리엇 / 유료 PDF·템플릿 / SaaS 툴 / 데이터 API / 강의 / 커뮤니티 / 스폰서십

**시나리오 추정 매트릭스**
| 시나리오 | 1년 보수 | 3년 현실 | 3년 낙관 |
|---|---|---|---|
| **10h/week 1인** | 월매출·주 수익원·구독자 수 | 동일 | 동일 |
| **2~3인 확장** | 동일 | 동일 | 동일 |

총 6개 셀. 각 셀에 월매출 추정, 주 수익원 2~3개, 필요 구독자/유저 수, 주요 가정.

**Unit Economics 하위 섹션** (Approach 3 흡수)
상위 3개 유망 모델에 대해: CAC 추정 · LTV 추정 · 회수 기간 · 손익분기점 유저 수.

**출력**
- 매핑 적합도 히트맵 (9 × ~10)
- 6셀 시나리오 매트릭스
- 상위 3모델 unit economics 테이블
- 추천 "첫 수익 엔진" 1개 + 근거

**분량**: ≤1000 단어 (가장 긴 축)

### 축 D — 피벗 옵션 조사

**핵심 질문**
> fcc가 현 3자산 중 어느 걸 버리고·키우고·신규 추가해야 "진짜 독립 사업"이 되는가?

**검토 피벗 후보 (브레인스토밍 범위)**
- 공모주 DB → 유료 API · B2B 금융 데이터 서비스
- 비용 최적화 → 월 구독형 "개인 재무 코치" SaaS
- 투자 캘린더 → 기관·증권사 화이트라벨
- 브랜드·도메인·GA4만 남기고 **"한국 40대 1인 가구 재무 포털"**로 완전 재정의 (Cycle 1 빈자리 흡수)
- 3자산 묶어서 B2B2C 기업 복지 제휴 (기업 내 1인 가구 직원용 포털)

**출력**
- 피벗 후보 3~5개 × (실현 난이도 / 예상 천장 / 법적 리스크 / 10h 가능 여부)
- 각 후보에 대한 "피벗 권장 여부 + 이유" 명시 의견

**분량**: ≤700 단어

## 7. 통합 섹션 (주조정자 작업)

4개 축 결과가 나온 후 주조정자는 다음을 수행:

1. **불일치 조정**: 축 간 가정·추정이 충돌하면 명시적으로 기록하고 기준 선택
2. **통합 원칙 6개** (Cycle 2 구조와 동일): 답지 형태로 여태경이 결정에 쓸 수 있는 형태
3. **시나리오 요약표**: 6셀 매트릭스를 한 눈에
4. **다음 사이클 후보 질문** 업데이트
5. **리서치 공백** 명시 (공개 데이터 없는 항목, 가설 비중)
